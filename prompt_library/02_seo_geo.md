# SEO & GEO Prompts

---

## 1. Content Brief (SEO)

**When to use:** Before writing any blog post or landing page targeting a keyword.

```
Create an SEO content brief for the topic: [TOPIC / TARGET KEYWORD]

Include:
- Target keyword + 5 semantic/related keywords to weave in
- Recommended title (H1)
- Suggested H2 structure (outline)
- Search intent: what is the reader actually trying to accomplish?
- What the top-ranking content covers that we must include
- Anything we can do differently to stand out
- Recommended word count range
- Internal linking suggestions (if I give you our site structure)

Audience: [DESCRIBE TARGET READER]
Company: [COMPANY NAME + ONE LINE ON WHAT THEY DO]
```

## 2. GEO Optimization Rewrite

**When to use:** Taking existing content and making it more likely to be cited in AI-generated answers.

```
I want to optimize this content to appear in AI-generated answers (ChatGPT, Claude, Perplexity).

Here is the original content:
[PASTE CONTENT]

Rewrite it so that:
1. The first 150 words directly and completely answer the main question
2. Key facts and claims are stated clearly and citeably
3. Structure uses clear headers and extractable sections
4. It reads as authoritative and specific (no filler language)
5. Any statistics or data points are preserved and highlighted

Keep the core message. Improve the structure and directness.
```

## 3. AI Visibility Audit Prompt

**When to use:** Testing how a brand currently appears in LLM responses.

```
I'm auditing how [BRAND NAME] appears in AI-generated answers.

Please answer these questions as if you're a potential B2B buyer:
1. "What are the best [PRODUCT CATEGORY] tools for [USE CASE]?" — is [BRAND] mentioned?
2. "What do people say about [BRAND NAME]?" — what comes up?
3. "What are alternatives to [BRAND NAME]?" — who do you compare them to?
4. "Why would someone choose [BRAND] over [COMPETITOR]?"

After each answer, note: how prominently is the brand featured? What signals (reviews, content, mentions) seem to be driving or limiting visibility?
```

*Run this same audit in ChatGPT, Perplexity, and Gemini separately and compare results.*

## 4. Keyword Cluster Builder

**When to use:** Building a topic cluster for a client's content plan.

```
I'm building a keyword cluster for [TOPIC] targeting [AUDIENCE TYPE] in [INDUSTRY].

Give me:
- 1 pillar topic (broad, high-volume)
- 5-8 cluster topics (more specific, supporting the pillar)
- For each cluster topic: the likely search intent and a suggested content format (blog, landing page, FAQ, comparison, etc.)

Think about how these pages would link to each other to build topical authority.
```

## 5. Meta Title & Description Generator

**When to use:** Writing or improving meta tags for SEO pages.

```
Write 3 options for a meta title and meta description for this page:

Page topic: [TOPIC]
Target keyword: [KEYWORD]
Audience: [WHO IS READING THIS]
Goal of the page: [WHAT DO YOU WANT THEM TO DO]

Rules:
- Title: under 60 characters, include keyword naturally
- Description: under 155 characters, include a clear value prop and soft CTA
- Avoid clickbait. Sound credible and specific.
```
